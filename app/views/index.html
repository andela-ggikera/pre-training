Welcome to the Simple Rest API

A simple RESTFUL Api for simple people!

Instruction manual

You can either do a GET, POST, PUT, OR DELETE http verb on this API.

This lets you have a CRUD functionality on your hands!

  • GET:get all the names stored in mongolab by append /simple after api: http:url/api/simple get all the names stored in mongolab
  • POST: append simple and parameter
  • PUT: update an already existing record
  • DELETE: add the id of the name you want to delete: http:/url/api/simple/23423424422
