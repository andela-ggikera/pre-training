Welcome to the Simple Rest API

A simple RESTFUL Api for simple people!

Instruction manual

You can either do a GET, POST, PUT, OR DELETE http verb on this API.

This lets you have a CRUD functionality on your hands!

  • GET: get all the names in the document: http:url/api/simple/
  • GET: get a specific name using its id: http:/url/api/simple/"id"
  • POST: append simple and parameter: http:/url/api/simple/"name"
  • PUT: update an already existing record: http:/url/api/simple/"id"
  • DELETE: add the id of the name you want to delete: http:/url/api/simple/"id"
